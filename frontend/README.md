# Frontend

Browser game client workspace built with Vite, TypeScript, and Phaser 3.

## Scripts

- `npm run dev --workspace frontend`: start Vite on `0.0.0.0:8080`.
- `npm run build --workspace frontend`: type-check and build the frontend.
- `npm run lint --workspace frontend`: run ESLint.
- `npm run format --workspace frontend`: check Prettier formatting.
